# 示例 Spec：用户认证 API

> 这是一个示例 spec 文件，展示 `/spec` 命令的用法。
> 你可以参考此格式编写自己的 spec。

## 概述

为项目添加用户认证功能，支持注册、登录和 token 验证。

## 功能需求

1. **注册**：接受 email + password，创建用户
2. **登录**：验证凭据，返回 JWT token
3. **验证**：中间件检查请求中的 token

## 约束

- 使用 bcrypt 加密密码
- JWT 过期时间 24 小时
- 不使用外部数据库，使用内存存储（演示用途）

## 验收标准

- [ ] 注册成功返回 201，重复 email 返回 409
- [ ] 登录成功返回 200 + token，失败返回 401
- [ ] 受保护路由无 token 返回 401
- [ ] 所有端点有单元测试
